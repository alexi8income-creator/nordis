Контекст

Маленькая обжарочная в Бергене с пятью прямыми контрактами с фермами. Сильный продукт, никакого присутствия онлайн. Нужна была витрина, в которой философия обжарки и подписка работают как одно повествование.

Запрос: премиальный голос без пафоса, фокус на ремесле, ощущение «арктики» — но тёплой, осязаемой, с ароматом кофе.

Цели

  • 01Объяснить происхождение зерна и ценность direct trade за 30 секунд скролла
  • 02Показать ремесло обжарки кинематографично, без видео
  • 03Конструктор подписки с конверсией ≥ 4%
  • 04Минимум зависимостей — один HTML, никакой CMS
[ 03 / СИСТЕМА ]

Система: тёплое бумажное против холодного графита.

[ 04 / РЕШЕНИЯ ]

Шесть интерактивных опор всего лендинга.

[ 01 ]

Hero с живым паром

Пять SVG-кривых пересчитываются 60 раз в секунду со сдвинутыми фазами. Без видеофайла — 0 KB трафика, идеально на мобильных. Параллакс уводит чашку при скролле.

SVG · requestAnimationFrame
[ 02 ]

Карта пяти широт

Абстрактная карта с линиями от Бергена до фермерских регионов. Hover синхронизирует точку и строку списка — и наоборот. Подача direct-trade без длинных текстов.

SVG · React state
[ 03 ]

Горизонтальный sticky-сторителлинг

Шесть стадий обжарки разворачиваются вбок при вертикальном скролле. Цвет зерна эволюционирует от зелёного до тёмного. Прогресс-бар внизу пин-секции — как часы обжарщика.

Sticky pin · IntersectionObserver
[ 04 ]

Drag-карусель сортов

Пять профилей: Aurora, Fjord, Polaris, Solstice, Tundra. На hover карточка переворачивается чёрной стороной с анимированными барами вкуса. Курсор меняется в режим drag.

Pointer events · CSS bars
[ 05 ]

Конструктор подписки

Три параметра — объём, частота, стиль. Цена пересчитывается живо, итог уезжает в drawer-корзину со слайд-анимацией. Все состояния — реальный React state.

React useState · live pricing
[ 06 ]

Магнитный курсор

Точка-следящая + кольцо с инерцией. Все интерактивные элементы помечены data-magnet — кольцо вырастает и меняет цвет на охру. Для не-десктопа курсор отключается.

RAF · linear interpolation
[ 05 / ПРОЦЕСС ]

Четыре недели — от брифа до запуска.

WEEK 01 · DISCOVERY

Бриф и дегустация

Поездка в Берген. Дегустация партий, разговор с обжарщиком Sigrid. Сбор языка бренда, формулировка манифеста.

WEEK 02 · DIRECTION

Дизайн-система

Финализация палитры, гарнитур, моно-меток. Несколько направлений hero — выбран сдержанный split с серифным гротеском.

WEEK 03 · BUILD

Hi-fi прототип

Сборка интерактивного прототипа на React + Babel. Параллельно — конструктор подписки и тестирование UX корзины.

WEEK 04 · POLISH

Доводка и QA

Адаптив, доводка анимаций, проверка горизонтального sticky на разных вьюпортах, оптимизация пара под mobile.

[ 06 / ТЕХНИКА ]

Один HTML, ноль билд-шагов.

STACK

  • React 18 (UMD) + Babel Standalone — без bundler
  • Чистый CSS, кастомные свойства, без фреймворков
  • Шрифты — Google Fonts (Instrument Sans/Serif, JetBrains Mono)
  • SVG для всей графики и анимаций

МОТИОН

  • requestAnimationFrame для пара и курсора
  • CSS transitions с cubic-bezier(.2,.8,.2,1)
  • Sticky-pin горизонтального скролла на чистом scroll-event
  • IntersectionObserver для reveal-паттернов

UX

  • Магнитный курсор отключается на <760px
  • Cart drawer + toast как единая система фидбэка
  • Hover-states у каждой кликабельной зоны
  • Адаптив 980px / 760px / 600px брейкпоинты
